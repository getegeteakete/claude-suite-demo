# CLAUDE SUITE - クラウド型顧客管理AIシステム

IT導入補助金2025申請用デモサイト

## 🎯 概要

**CLAUDE SUITE**は、Claude AIを活用したクラウド型顧客管理システムです。
顧客対応、販売支援、経営分析など、複数の業務プロセスをAIで効率化します。

**提供企業:** 合同会社UMA

## 🚀 使い方

### ローカルで閲覧

1. このリポジトリをクローン:
```bash
git clone https://github.com/YOUR_USERNAME/claude-suite-demo.git
cd claude-suite-demo
```

2. `index.html` をブラウザで開く

### GitHub Pagesで公開

1. リポジトリの Settings → Pages に移動
2. Source を `main` ブランチに設定
3. 公開URLでアクセス可能
